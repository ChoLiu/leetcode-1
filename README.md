18. 4Sum
    - 套用之前用过的3Sum，再加一层即可。再加层的时候有比较巧妙的写法：
    - 3/2 重写完毕。

``````python
    def fourSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[List[int]]
        """
        nums.sort()
        res = []
        for idx in range(len(nums)-3):
            if idx != 0 and nums[idx] == nums[idx-1]:
                continue
                
            for ite in self.threeSum(nums[idx+1:],target-nums[idx]):
                res.append([nums[idx]]+ite)
        return res
``````
